How to use OrbitalVis

Prerequisites

OrbitalVis requires the following software to be installed on your computer:

  • Sun/Oracle Java Runtime Environment 1.6, update 18 or later. Earlier updates of the JRE might work, but haven't been tested. JRE's other than Sun's/Oracle's might also work, but haven't been tested. Oracle's latest JRE is a free download, so this shouldn't pose a real problem.
  • JOGL [[HAVE TO EXPERIMENT]]

Using OrbitalVis

Selecting a reaction mechanism

You can select a reaction mechanism by editing the file generic-props.txt with a text editor (Notepad, for example). In the part of that file marked "Which model?", you can remove the # character at the beginning of the line that indicates the model you want. So, for example, if you want to see an E1 reaction, you change the line that looks like this:
#model: E1
to this:
model: E1
Make sure that the lines for all the other models start with a #, and you're ready to run.

If you want to see an SN2 reaction, just make sure that lines for all the other models start with a # character, and you're ready to go. (SN2 is the default, which is why there's no specific line for it.)

Many other properties of the animation can also be changed by editing either the generic-props.txt file or the several mechanism-specific files (for example, E1-props.txt sets properties specific to the E1 animation).

Starting the program

Just open OrbitalVis.jar.

Controls

Running the reaction

The reaction can be run by using the arrow buttons at the top (press and hold to run the reaction forward or backward). A specific point in the reaction can be chosen with the slider bar between the two arrow buttons.

Changing the viewpoint

The Rock button makes the model rock back and forth automatically. This motion can be quite helpful in disambiguating the 3D structure, especially with more complex reactions. Clicking the same button again (it will now be marked Stop) makes the rocking stop.

The model can also be rotated about the vertical axis by clicking the mouse in the 3D window and dragging sideways. This can be useful if you want to look at a reaction from some particular angle.

There is currently no way to rotate the model about either of the other two axes.

Attacking from the other side

If the reaction mechanism allows attack from the other side (such as E1 or SN1), the Attack other side button will restart the simulation with the attacking group coming from the other side.

Exiting the program

Closing the window exits the program. (For example, on Windows, click the little button with the “X” in the upper-right corner.)
